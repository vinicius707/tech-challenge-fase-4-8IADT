# UI: Next.js + Tailwind + shadcn/ui

O frontend Limen usa Tailwind CSS e primitives shadcn/ui (tabelas, dialogs, forms, badges) sobre Next.js App Router. Motivo: velocidade para painéis clínicos/operacionais com acessibilidade básica, sem visual de admin-template genérico. Alternativas rejeitadas: CSS puro (mais lento); MUI/Chakra (cara de template).
