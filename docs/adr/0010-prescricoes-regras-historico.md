# Anomalias de prescrição: regras + desvio longitudinal

Anomalias de Prescrição combinam regras determinísticas (dose fora de faixa, intervalo irregular, medicamento sem histórico) com desvio longitudinal quando o Paciente já tem Casos anteriores. Motivo: demo funciona com um único Caso via regras; o histórico por Paciente reforça o escopo robusto. Alternativa rejeitada: apenas ML sem regras (pouco explicável na Justificativa).
